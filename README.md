# Milkyway

Milkyway is playground for discovering:

- Linux development
- ASP Net Core
- Docker containers
- Kubernetes
- Istio

## Setup
- Install vscode
- Install aspnetcore sdk
- Install Docker
- Install Kubernetes CLI
- Install Minikube (kubernetes cluster to be run on laptop)
- Install Istio

## Docker
